📚 СОДЕРЖАНИЕ
-------------

1. ДАШБОРД
   ----------
   Обзор всей системы в реальном времени
   - Общая выручка и прибыль за месяц
   - Активные фабрики и количество сотрудников
   - Средняя эффективность и уровень брака
   - Графики производства за последние 7 дней
   - История прибыли по месяцам
   - Последние уведомления
   - Производительность каждой фабрики

2. ПРОИЗВОДСТВО
   -----------
   Управление заказами и производством
   - Все заказы с прогресс-баром
   - Статусы: в ожидании, в работе, завершен, задержан
   - Активные производственные партии
   - Стадии: раскрой, шитье, контроль качества, упаковка
   - Обзор мощностей всех фабрик
   - Количество работников и брака

3. СОТРУДНИКИ
   ----------
   Управление персоналом
   - Список всех сотрудников
   - Показатели эффективности каждого
   - Статистика произведенных единиц
   - Расчет заработной платы
   - Бонусы за выполнение KPI
   - Рейтинг лучших сотрудников
   - Фильтрация по фабрикам
   - Сортировка по эффективности, зарплате

4. ФИНАНСЫ
   --------
   Финансовый контроль
   - Выручка, расходы, прибыль
   - Рентабельность бизнеса
   - Денежные потоки (cash in/out)
   - История транзакций
   - Просмотр по дням или месяцам
   - Финансовый результат по фабрикам
   - Графики доходов и расходов

5. АНАЛИТИКА
   ----------
   Аналитика и отчеты
   - Тренды производства
   - Отслеживание уровня брака
   - Сравнение эффективности фабрик
   - Круговые диаграммы (приоритеты заказов, статусы)
   - Рейтинг лучших сотрудников
   - Общий показатель здоровья (88%)

⚠️ ОГРАНИЧЕНИЯ ДЕМО-ВЕРСИИ
---------------------------
- Используются тестовые данные (mock data)
- Нет подключения к реальной базе данных
- Нет обновлений в реальном времени
- Один пользователь (admin)
- Данные не сохраняются при перезагрузке

💡 СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ
---------------------------
1. Для входа используйте данные из файла ЛОГИН.txt
2. Переключайтесь между разделами через левое меню
3. Используйте фильтры и сортировку в таблицах
4. Наведите на графики для просмотра деталей
5. Для выхода нажмите "Выйти" в левом меню
